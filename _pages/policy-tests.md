      <h3><span class="lang-en">Written (Closed-Book) Portions</span><span class="lang-ko">필기 (닫힌 책) 부분</span></h3>
      <p><span class="lang-en">No phones, notes, AI tools, or resources of any kind. You must work completely independently.</span><span class="lang-ko">휴대폰, 노트, AI 도구 등 어떤 자료도 사용할 수 없습니다. 완전히 독립적으로 작업해야 합니다.</span></p>

      <h3><span class="lang-en">Programming (Open-Book) Portions</span><span class="lang-ko">프로그래밍 (열린 책) 부분</span></h3>
      <p><span class="lang-en">You may use any reference — including the Internet and AI tools — but <strong>you must write all code yourself</strong>. Pasting AI output directly is not allowed. Code copied from ChatGPT often does not function properly and will not receive full credit.</span><span class="lang-ko">인터넷과 AI 도구를 포함한 모든 자료를 참고용으로 사용할 수 있지만, <strong>모든 코드는 직접 작성해야 합니다</strong>. AI 출력을 직접 붙여넣는 것은 허용되지 않습니다.</span></p>

      <h3><span class="lang-en">General Rules</span><span class="lang-ko">일반 규정</span></h3>
      <ul>
        <li><span class="lang-en">No cellphones during any test portion.</span><span class="lang-ko">시험 중 휴대폰 사용 금지.</span></li>
        <li><span class="lang-en">No bathroom breaks during tests.</span><span class="lang-ko">시험 중 화장실 이용 금지.</span></li>
        <li><span class="lang-en">No talking. Cheating = zero for that test.</span><span class="lang-ko">대화 금지. 부정행위 = 해당 시험 0점.</span></li>
        <li><span class="lang-en">Regrading requests must be submitted by email; regrading may result in score going up or down.</span><span class="lang-ko">재평가 요청은 이메일로 제출; 재평가 결과 점수가 오르거나 내릴 수 있습니다.</span></li>
      </ul>
