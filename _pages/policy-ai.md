<h3 id="overview">Overview 개요</h3>

AI tools like ChatGPT, GitHub Copilot, and similar generative AI systems are powerful but imperfect. This policy explains when and how you may use them in this course.
<br>ChatGPT, GitHub Copilot 등의 AI 도구는 강력하지만 완벽하지 않습니다. 이 정책은 수업에서 AI 도구를 언제, 어떻게 사용할 수 있는지 설명합니다.

The goal of this course is for **you** to develop skills and understanding. AI tools used as a shortcut — rather than a learning aid — undermine that goal and your own development.
<br>이 강좌의 목표는 **여러분**이 기술과 이해를 개발하는 것입니다. 학습 보조 도구가 아닌 지름길로 AI 도구를 사용하는 것은 그 목표와 여러분 자신의 발전을 저해합니다.

---

<h3 id="allowed">What IS Allowed 허용 사항</h3>

You **may** use AI tools for:
<br>다음과 같은 목적으로 AI 도구를 **사용할 수 있습니다**:

- **Understanding concepts:** Asking ChatGPT to explain a concept in simpler terms, or in Korean, is allowed and encouraged.<br>**개념 이해:** ChatGPT에게 개념을 더 쉬운 용어로, 또는 한국어로 설명해 달라고 요청하는 것은 허용되며 권장됩니다.
- **Debugging help:** Using AI to help understand why your code has an error — as long as *you* understand and write the fix yourself.<br>**디버깅 도움:** 코드에 오류가 있는 이유를 이해하는 데 AI를 사용하는 것은 허용됩니다 — 단, *여러분*이 스스로 수정 사항을 이해하고 작성해야 합니다.
- **Brainstorming:** Using AI to generate ideas for a project or paper topic, which you then develop yourself.<br>**브레인스토밍:** 프로젝트나 논문 주제에 대한 아이디어를 생성하는 데 AI를 사용하는 것은 허용됩니다 — 이후 스스로 발전시켜야 합니다.
- **Translation:** Using AI to help translate between English and Korean.<br>**번역:** 영한 번역을 돕는 데 AI를 사용하는 것은 허용됩니다.
- **Open-book test portions:** During programming sections of tests (where open-book is explicitly permitted), AI tools may be used as a reference — but you must write all code yourself.<br>**오픈북 시험 부분:** 시험의 프로그래밍 섹션 (오픈북이 명시적으로 허용되는 경우) 에서 AI 도구는 참고용으로만 사용할 수 있습니다 — 단, 모든 코드는 직접 작성해야 합니다.

---

<h3 id="not-allowed">What is NOT Allowed 금지 사항</h3>

You **may NOT** use AI tools to:
<br>다음과 같은 목적으로 AI 도구를 **사용할 수 없습니다**:

- **Generate code you submit as your own** for assignments or projects.<br>과제나 프로젝트에서 **자신의 것으로 제출하는 코드를 생성**하는 데 사용할 수 없습니다.
- **Write or substantially rewrite** essays, reports, or written assignments.<br>에세이, 보고서, 또는 작성 과제를 **작성하거나 실질적으로 재작성**하는 데 사용할 수 없습니다.
- **Solve practice exercises** that are meant to build your skills during class.<br>수업 중 기술 향상을 위한 **실습 문제를 풀기** 위해 사용할 수 없습니다.
- **Any use during closed-book portions** of tests or exams.<br>시험이나 시험의 **클로즈북 부분** 에서는 어떤 사용도 금지됩니다.

---

<h3 id="assignments">Assignments & Projects 과제 및 프로젝트</h3>

**Why AI-generated code often fails:**
AI-generated code frequently contains errors, uses deprecated syntax, or does not fit the exact requirements of an assignment. **Code copied from ChatGPT often does not function properly and will not receive full credit.**
<br>**AI 생성 코드가 종종 실패하는 이유:**
AI 생성 코드는 오류를 포함하거나, 더 이상 사용되지 않는 구문을 사용하거나, 과제의 정확한 요구 사항에 맞지 않는 경우가 많습니다. **ChatGPT에서 복사한 코드는 종종 제대로 작동하지 않으며 완전한 점수를 받지 못할 수 있습니다.**

**Evidence of understanding is required:**
If you submit code that you cannot explain or demonstrate during class, you may receive a zero for that assignment. I reserve the right to ask any student to explain their submitted work.
<br>**이해의 증거가 필요합니다:**
수업 중에 설명하거나 시연할 수 없는 코드를 제출하면 해당 과제에 대해 0점을 받을 수 있습니다. 저는 모든 학생에게 제출한 작업을 설명하도록 요청할 권리를 보유합니다.

---

<h3 id="tests">Tests & Exams 시험</h3>

- **Written (closed-book) portions:** No AI tools allowed. No phone, no ChatGPT, no Copilot.<br>**작성형 (클로즈북) 부분:** AI 도구 사용 금지. 휴대폰, ChatGPT, Copilot 모두 금지.
- **Programming (open-book) portions:** You may use any resource including the Internet and AI for *reference* — but **you must write all code yourself.** Pasting AI output directly is not allowed.<br>**프로그래밍 (오픈북) 부분:** 인터넷 및 AI를 포함한 모든 자료를 *참고용*으로 사용할 수 있습니다 — 단, **모든 코드는 직접 작성해야 합니다.** AI 출력을 직접 붙여넣는 것은 허용되지 않습니다.

---

<h3 id="disclosure">Disclosure Requirement 공개 요건</h3>

If you used an AI tool in completing any assignment, **you must disclose this** in a comment at the top of your file or in your submission notes. For example:
<br>과제를 완료하는 데 AI 도구를 사용한 경우, **이를 공개해야 합니다** — 파일 상단의 주석 또는 제출 메모에. 예시:

```
# AI Disclosure: Used ChatGPT to understand the difference between
# list comprehension and generator expressions. All code written by me.
```

Failure to disclose AI use when it occurred is considered academic dishonesty and will be treated as such.
<br>AI 사용이 있었을 때 이를 공개하지 않는 것은 학문적 부정직으로 간주되어 그에 따라 처리됩니다.
